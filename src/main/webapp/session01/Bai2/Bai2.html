Cấu trúc thư mục của dự án

.gradle: Thư mục nội bộ Gradle để cache và xử lý các tác vụ.

.idea: Cấu hình dành cho IntelliJ IDEA.

build: Thư mục chứa output khi build (được Gradle tạo ra).

java: Chứa class Java của bạn

resources: Chứa file cấu hình như .xml, .properties,...

src/main: Chứa mã nguồn chính và tài nguyên.

webapp: Thư mục gốc của tài nguyên web (HTML, JSP, CSS, JS,...).

src/test: Nơi chứa mã kiểm thử (test code).

build.gradle: Định nghĩa cách Gradle build dự án (thư viện, plugin, servlet api...).

settings.gradle: Định nghĩa tên root project.
